# E-COMMERCE COM REACT

Este é um projeto de E-commerce desenvolvido utilizando **React**, **Redux** para gerenciamento de estado e **React Router** para navegação.

## 🚀 Funcionalidades
* Listagem de produtos.
* Carrinho de compras funcional (Adicionar/Remover).
* Cálculo de preço total.
* Navegação entre páginas (Home e Carrinho).

## 🛠️ Tecnologias Utilizadas
* React.js
* Redux (para o estado global do carrinho)
* React Router Dom (para rotas)
* CSS Customizado
